Инструкция по использованию интерпретатора:

1. Нажимайте на инструкции по порядку для их выполнения

2. После выполнения всех инструкций появится готовая жалоба

3. Каждая инструкция изменяет состояние процессора (EIP)

4. INT 0x70 - специальное прерывание для генерации жалобы в Роскомнадзор
